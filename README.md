# Introduction

`big-ts` is a port of the [big.js](https://mikemcl.github.io/big.js/) library, written in TypeScript using immutable and functional principles.

# Installation

```
npm install big-ts
```

# Documentation

Docs can be found [here](https://adameier.github.io/big-ts).
